07 / GET STARTED

Build your first AdvanxJS app

From zero to a running component in under two minutes.

Installation

Five commands from empty folder to a running app.

1 Scaffold
shell 
$ npx advanxjs create my-app

Copied to clipboard!

2 Install
shell 
$ cd my-app && bun install

Copied to clipboard!

3 Add a component
shell 
$ advanx add navbar

Copied to clipboard!

4 Build
shell 
$ advanx build src/pages

Copied to clipboard!

5 Serve
shell 
$ advanx serve dist

Copied to clipboard!

The Trinity

Every component is exactly three files. No exceptions. This is the Counter example.

logic.ts

import { signal } from "advanxjs";

export const count = signal(0);

export function increment() {
  count.value++;
}

view.html

<p>Count: {{ count }}</p>

<button ax-on:click="increment">
  Add
</button>

style.css

p {
  font-size: 1.5rem;
  font-weight: 700;
  color: var(--ax-brand);
}

button {
  cursor: pointer;
}
AI Feature

advanx explain

The advanx explain command reads your component's .advanx-meta.json and sends a structured map — signals, actions, bindings — directly to your AI assistant. No hallucinations. No guessing.

shell 
$ advanx explain src/components/counter

# Component: counter
# Signals:   count (number, default: 0)
# Actions:   increment()
# Bindings:  {{ count }}, ax-on:click="increment"
