Build a Task List in 5 Minutes

Afraid of learning a new framework? Don't be. AdvanxJS is just the HTML and TypeScript you already know. Follow these four steps and you'll have a reactive task list running — with zero magic.

1 Scaffold the app
shell 
$ npx advanxjs create my-app

Copied to clipboard!

No hooks. No boilerplate. Just logic.

2 Write the logic — logic.ts

One signal holds the array of tasks. One function adds to it. That's the whole brain.

logic.ts 
import { signal } from "advanxjs";

export const tasks = signal(["Learn AdvanxJS"]);
export const draft = signal("");

export function addTask() {
  if (!draft.value.trim()) return;
  tasks.value = [...tasks.value, draft.value];
  draft.value = "";
}

Copied to clipboard!

No hooks. No boilerplate. Just logic.

3 Write the view — view.html

Plain HTML. ax-model binds the input, ax-for loops the list. No JSX, no render functions.

view.html 
<input ax-model="draft" placeholder="New task..." />
<button ax-on:click="addTask">Add</button>

<ul>
  <li ax-for="task in tasks">{{ task }}</li>
</ul>
